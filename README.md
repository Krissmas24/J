# J - Fisch Macro Rebirth

A high-performance, aesthetically pleasing fishing macro for Roblox Sober on Linux.

## Features
- Yin-Yang (Black/White/Gold) Theme
- SQLite Persistence & Event Tracking
- Wayland/Hyprland Support (grim + ydotool)
- Modular Architecture

## Tracking & Checkpoints
This macro tracks every event and maintains checkpoints for system recovery and progress analysis.
